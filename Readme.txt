"Study1_processed_data.csv" [see Data_Explainer.txt for more information]
CSV file containing all pre-processed long-form data for Study 1

"Study2_processed_data.csv" [see Data_Explainer.txt for more information]
CSV file containing all pre-processed long-form data for Study 2

"NComms_Main_Final.R"
R script containing all analyses reported in the main manuscript (except continuous eyetracking & model fitting).

"EyetrackingAnalyses.zip"
Compressed folder containing continuous eyetracking analyses and permutation cluster-correction.

"ComputationalModels.zip"
Compressed folder containing computational model scripts and results (fitting & cross-validation)

"Figures.zip"
Compressed folder containing all figures in the main manuscript in pdf format.

"SupplementaryAnalyses.zip"
Compressed folder containing all supplementary analyses & generated supplementary figures described in the Supplementary Notes.

"SubjectData.zip"
Compressed folder containing individual folders for subject data collected using PsychToolBox in MATLAB for Study 1
